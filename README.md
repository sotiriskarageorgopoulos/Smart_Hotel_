# Smart Hotel

## Run Locally

Clone the project

```bash
  git clone --branch front-end https://github.com/sotiriskarageorgopoulos/Smart_Hotel.git  
```

Go to the project directory & install dependencies

```bash
  npm install
```

Start the server at the port 3000

```bash
  npm run start
```
